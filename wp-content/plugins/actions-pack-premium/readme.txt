== Description ==

Actions Pack extends Elementor Pro's default Form widget providing different form actions those are
executed when an Elementor form is submitted from the frontend. One such important action is Elementor User Registration
that will help you create unlimited Wordpress users with the power of Elementorâ€™s Native Form Widget. Currently the
Plugin includes 5 premium actions and counting.
